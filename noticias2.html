Já há uma ferramenta que ajuda a instalar o Windows 11 em computadores mais antigos.

A versão 3.16 Beta 2 do software Rufus permite adaptar o Windows 11 para computadores que não cumprem os requisitos de instalação definidos pela Microsoft, como a existência de um chip TPM. A Rufus é uma ferramenta de criação de unidades de armazenamento, como pen USB, de arranque do sistema operativo. Agora, a versão mais recente do software inclui um ‘modo alargado’ que ajuda os utilizadores a ultrapassarem os requisitos de segurança e de RAM da Microsoft para o novo sistema operativo.

Para ter acesso ‘direto’ ao Windows 11, através do sistema oficial de atualização integrado no Windows 10, é necessário ter um computador que cumpra os seguintes requisitos mínimos: ter um dos processadores da AMD, Intel e Qualcomm oficialmente suportados; 64 GB de armazenamento; 4 GB de memória RAM; um chip TPM 2.0; e ter uma placa gráfica que seja compatível com DirectX 12 e o Windows Display Driver Model 2.0.

O TPM – de Trusted Plataform Module – é um componente integrado nas placas-mãe (motherboard) e que tem como objetivo conferir maior segurança ao arranque dos computadores e a algumas aplicações. Para instalar o Windows 11, é necessário o computador ter TPM 2.0 – isto significa que só computadores comprados, sensivelmente, a partir de 2016 é que garantem esta compatibilidade. A Microsoft diz que o Windows 11 funciona também em computadores que têm chips TPM mais antigos (versão 1.2), mas que não recomenda a instalação do novo sistema operativo nestas máquinas.

A atualização para a versão beta mais recente do Rufus permite aos utilizadores instalarem o sistema em quase todos os computadores produzidos na última década e inclusive nos computadores que não têm BIOS UEFI, noticia a publicação Tom’s Hardware.

Os utilizadores interessados em proceder à instalação só têm de descarregar uma imagem ISO do Windows 11 com a Microsoft e a aplicação Rufus 3.16 Beta 2, seguindo os passos indicados na página do GitHub.

De recordar, no entanto, que a Microsoft não vai disponibilizar atualizações de segurança para os utilizadores de computadores antigos que decidam instalar o Windows 11.
