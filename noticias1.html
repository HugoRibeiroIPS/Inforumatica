A Microsoft lança apps Android para Windows 11 em fevereiro. A nova atualização vai trazer ainda novidades às aplicações do Windows, como o Media Player e o Notepad. A Microsoft vai lançar uma preview gratuita do Windows 11 com aplicações Android para a Microsoft Store já em fevereiro. Ainda não se sabe quantas serão, mas as aplicações podem ser encontradas na Amazon Appstore.

Esta versão do sistema operativo conta também com atualizações às aplicações do Windows. Por exemplo, a barra de tarefas vai incluir um botão de silenciamento de chamadas, partilhas de ecrã mais simples e as aplicações Media Player e Notepad serão renovadas. Uma das principais novidades prometidas para o Windows 11 foi o suporte para as aplicações Android, que vai ajudar o utilizador a usá-las no PC e vai despertar o interesse dos utilizadores em PCs Windows com ecrã tátil. A versão definitiva só chegará, contudo, uns meses após o lançamento desta nova versão do Windows 11.
